## 📖 프로젝트 소개
RUMUNE 는 이커머스 프로젝트의 일환으로 같은 상품이지만 다른 가격에 다른 게시글에서 판매되는 것에 대한 의문으로부터 시작되었습니다.
초기에는 개인이 제작한 핸드메이드 굿즈를 판매하는 것으로 시작하지만, 차후에는 커뮤니티 및 개인이 판매를 할 수 있는 소싱 플랫폼으로도 리워크 가능하도록
확장에 열려있는 유연한 코드를 작성하는데 집중한 프로젝트입니다.
